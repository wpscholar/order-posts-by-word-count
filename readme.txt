== Description ==

The *Order Posts by Word Count* plugin creates a widget that lists posts in order by word count.  Posts can be sorted in ascending or descending order and you can select how many posts you want to show. The widget can be used in the sidebar or any widgetized area.

The widget uses your existing CSS rules to adapt to the look and feel of your site.  If the widget does not look quite right, your existing CSS files will likely need tweaking.  You can visit the [plugin homepage](http://www.wordpress-marketing.com/plugins/order-posts-by-word-count/ "Order Posts by Word Count Homepage") for questions regarding this plugin.

== Installation ==

= Prerequisites =
If you don't meet the below requirements, I highly recommend you upgrade your WordPress install or move to a web host that supports a more recent version of PHP.

* Requires WordPress version 4.3 or greater
* Requires PHP version 5.2 or greater ( PHP version 5.2.4 is required to run WordPress version 3.2 )

= Install via Search =

1. In your WordPress admin, go to 'Plugins' and then click on 'Add New'.
2. In the search box, type in 'Order Posts by Word Count' and hit enter.  This plugin should be the first and likely the only result.
3. Click on the 'Install' link.
4. Once installed, click the 'Activate this plugin' link.

= Install via Direct Download =

1. Download the .zip file containing the plugin.
2. In the WordPress admin, click on 'Plugins' and then 'Add New'
3. On the resulting 'Install Plugins' page, find the 'Upload' link at the top and click on it.
4. Click on the 'Choose File' button and point your machine to the .zip you initially downloaded.
5. Click 'Install Now'
6. After the install is complete, click the 'Activate Plugin' link.

== Frequently Asked Questions ==

= How do I use this plugin? =

1. In the WordPress admin, click on ‘Appearance’ in the left-hand navigation and then on ‘Widgets’ in the submenu.
2. Locate the widget titled ‘Order Posts by Word Count’ and drag it into your sidebar or other widget area.
3. Configure your settings and click 'Save'
4. Visit the front end of the site to view.
